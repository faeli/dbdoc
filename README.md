DBDoc
=====

DBDoc is a simple database doc build to one html file.

Installing
----------
Install and update using [`pip`][0]:
```bash
pip install -U DBDoc
```

A Simple Example
----------------
```bash
dbdoc mysql+pymysql://root:root@127.0.0.1:3306/testing
```

[0]: https://pip.pypa.io/en/stable/quickstart/
